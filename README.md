## 3. 서비스 소개
네이버의 스포츠 뉴스를 스크래핑하여 클로바 요약 API로 각 뉴스를 요약해 보여줍니다.<br/>
회원들이 뉴스에 대한 의견을 댓글로 나눌 수 있고, 마음에 드는 뉴스를 북마크하거나 다른 회원의 댓글과 북마크를 모아볼 수 있습니다.<br/><br/>
[상세 정보 보기](https://github.com/2022-Harmony/NewsCommunity-bFinal)
